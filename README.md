# Yummies Starter Kit

WIP

## structure

* `build/` – output folder
* `conf/` – webpack configs
* `tasks/` – start tasks
* `src/index.js` – main entry point

## install

```
npm i
```

## usage

### dev

```
npm start
open http://localhost:3000/webpack-dev-server/
```

### build

```
npm run build
open build/index.html
```
