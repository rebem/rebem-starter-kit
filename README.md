WIP

## structure

* `build/` – output folder
* `conf/` – configs
* `tasks.js` – [start](https://github.com/start-runner/start) tasks
* `src/index.js` – main entry point

## install

```
npm i
```

## usage

### dev

```
npm start dev
open http://localhost:3000/webpack-dev-server/
```

### build

```
npm start build
open build/index.html
```
