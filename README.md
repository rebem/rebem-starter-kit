## Yummies Starter Kit

WIP

### structure

* `build/` – output folder
* `conf/` – configs
* `gulp/` – gulpo tasks
* `src/index.es6` – main entry point
* `src/components/` – current layer's components

### dev

```
npm run dev
open http://localhost:3000/webpack-dev-server/
```

### build

```
npm run build
open build/index.html
```
