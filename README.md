## Yummies Starter Kit

WIP

### structure

* `.yummies.yml` – main Yummies config
* `build/` – output folder
* `conf/` – webpack configs
* `tasks/` – start tasks
* `src/index.js` – main entry point
* `src/components/` – current layer's components

## install

```
npm i
```

### dev

```
npm run dev
open http://localhost:3000/webpack-dev-server/
```

### build

```
npm run build
open build/index.html
```
