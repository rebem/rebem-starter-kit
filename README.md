# Yummies Starter Kit

WIP

## structure

* `build/` – output folder
* `conf/` – webpack configs
* `tasks/` – start tasks
* `src/index.js` – main entry point
* `src/.yummies.yml` – current project layer's config

## install

```
npm i
```

## usage

### dev

```
npm run dev
open http://localhost:3000/webpack-dev-server/
```

### build

```
npm run build
open build/index.html
```
